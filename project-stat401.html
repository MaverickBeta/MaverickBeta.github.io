Overview

This project presents an end-to-end data pipeline and interactive web-based dashboard for analyzing social discourse propagation across Chinese social media platforms (Weibo and Xiaohongshu/Rednote). We focused on the SileMa app, engineering a complete workflow from web scraping unstructured social media data to building dynamic visualizations that reveal cross-platform engagement patterns, sentiment trends, and the geographical structure of digital public arenas.

Key Contributions

End-to-End Data Pipeline

  • Engineered a web-scraping pipeline using Python and Playwright to harvest unstructured social media posts, comments, and search cards from Rednote (Xiaohongshu).
  • Handled large-scale JSONL/CSV data cleaning and transformation to prepare raw social media data for downstream analysis.
  • Processed multiple data formats including Weibo posts, Weibo comments, and Xiaohongshu search cards with Chinese stopword filtering.

Natural Language Processing

  • Implemented NLP workflows leveraging SnowNLP for Chinese text segmentation, keyword extraction, and sentiment analysis.
  • Quantified public emotions and discourse trends over time across both Weibo and Xiaohongshu platforms.
  • Applied TF-IDF analysis to extract platform-specific keywords and identify discourse patterns unique to each social media ecosystem.

Interactive Dashboard Development

  • Designed and built a dynamic web-based dashboard using HTML/CSS/JavaScript with interactive time-filtering capabilities.
  • Developed temporal timeline visualizations for cross-platform engagement tracking.
  • Created user-topic bipartite network graphs to map relationships between discussion topics and user engagement.
  • Built geographic influence network visualizations showing the spatial distribution of digital discourse.
  • Implemented sentiment distribution plots and interactive word clouds for intuitive sentiment exploration.

Analytical Impact

Facilitated dynamic exploration of cross-platform (Weibo vs. Rednote) opinion formation, revealing platform-dependent discourse styles and the geographical structure of digital public arenas. The dashboard enables stakeholders to identify trending topics, sentiment shifts, and regional variations in public opinion propagation patterns.

项目概述

本项目搭建了一套端到端的数据处理流程和交互式Web仪表盘，用于分析中国社交媒体平台（微博和小红书）上的社交 discourse 传播。我们以SileMa App为研究对象，设计了从网络爬取非结构化社交媒体数据到构建动态可视化的完整工作流，揭示跨平台互动模式、情感趋势和数字公共领域的地理结构。

主要贡献

端到端数据流程

  • 使用Python和Playwright构建网络爬虫流程，从小红书（Xiaohongshu）采集帖子、评论和搜索卡片等非结构化社交媒体数据。
  • 处理大规模JSONL/CSV数据清洗和转换，为下游分析准备原始社交媒体数据。
  • 处理多种数据格式，包括微博帖子、微博评论和小红书搜索卡片，并应用中文停用词过滤。

自然语言处理

  • 使用SnowNLP实现NLP工作流，完成中文文本分词、关键词提取和情感分析。
  • 量化微博和小红书两个平台上的公众情绪和 discourse 趋势随时间的变化。
  • 应用TF-IDF分析提取平台特有关键词，识别各社交媒体生态系统的 discourse 模式。

交互式仪表盘开发

  • 使用HTML/CSS/JavaScript设计并构建了动态Web仪表盘，支持交互式时间过滤。
  • 开发了跨平台互动追踪的时间线可视化。
  • 创建了用户-话题二分网络图，映射讨论话题与用户参与之间的关系。
  • 构建了地理影响网络可视化，展示数字 discourse 的空间分布。
  • 实现了情感分布图和交互式词云，用于直观的情感探索。

分析影响

支持对跨平台（微博 vs. 小红书）舆论形成的动态探索，揭示了平台依赖的 discourse 风格和数字公共领域的地理结构。该仪表盘帮助利益相关者识别热门话题、情感变化和舆论传播的区域差异。
