SAE-Detoxification: Mechanistic Interpretability for LLM Safety

Sep 2025 – Present · With Dr. Lawrence Carin, Duke University
View on GitHub

Overview

This project applies Sparse Autoencoders (SAEs) to decode activation patterns in autoregressive language models (specifically Gemma-2-2B), achieving a mechanistic understanding of detoxification behaviors. By analyzing the Pareto Frontier between Toxicity Reduction (Safety) and Cross-Entropy Loss (Side Effects), we identify optimal features that offer the best trade-off for steering LLMs toward safer outputs without compromising general capabilities.

My Role: Research assistant in Dr. Lawrence Carin's lab. I contributed to the SAE-based feature analysis pipeline, assisted in validating the feasibility of the FST model, and participated in the full experimental cycle from data generation to evaluation.

Key Contributions

  • Applied Sparse Autoencoders to decode activation patterns in autoregressive language models (Gemma-2-2b), achieving mechanistic understanding of detoxification behaviors through feature-level analysis.
  • Analyzed the Pareto frontier between toxicity reduction and cross-entropy loss to identify steering features that minimize side effects while maximizing safety improvements.
  • Built and ran the end-to-end steering pipeline including feature discovery, generation, toxicity scoring, and trade-off visualization using BERT-based Detoxify and WikiText-2 CE loss metrics.
  • Generated publication-quality Pareto plots and per-feature trade-off visualizations.

Technical Details

  • Model: Gemma-2-2B with SAE probes on residual stream (Layer 20)
  • Feature Discovery: Parallel corpus differential analysis
  • Evaluation: BERT-based toxicity scoring (Detoxify) + WikiText-2 CE loss
  • Steering: Feature-level steering with configurable strength
  • Analysis: Pareto frontier computation and trade-off visualization

SAE 去毒化：大语言模型安全的机械可解释性

项目概述

本项目应用稀疏自编码器（SAE）解码自回归语言模型（Gemma-2-2B）中的激活模式，实现对去毒化行为的机械可解释性理解。通过分析毒性降低（安全性）与交叉熵损失（副作用）之间的帕累托前沿，我们识别出最优特征，在不对通用能力造成损害的前提下，引导大语言模型输出更安全的内容。

我的角色：Lawrence Carin教授实验室的研究助理。我参与了基于SAE的特征分析流程，辅助验证FST（特征引导去毒化）模型的可行性，并参与了从数据生成到评估的完整实验周期。

主要贡献

  • 应用稀疏自编码器解码自回归语言模型（Gemma-2-2b）中的激活模式，通过特征级分析实现对去毒化行为的机械可解释性理解。
  • 分析毒性降低与交叉熵损失之间的帕累托前沿，识别出副作用最小化且安全性最大化的引导特征。
  • 构建并运行端到端引导流程，包括特征发现、文本生成、毒性评分和权衡可视化，使用基于BERT的Detoxify和WikiText-2交叉熵损失指标。
  • 生成可用于发表的帕累托图和逐特征权衡可视化。

技术细节

  • 模型：Gemma-2-2B，残差流（第20层）SAE探针
  • 特征发现：平行语料库差异分析
  • 评估：基于BERT的毒性评分（Detoxify）+ WikiText-2交叉熵损失
  • 引导：特征级引导，可配置引导强度
  • 分析：帕累托前沿计算与权衡可视化
